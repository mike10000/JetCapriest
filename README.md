# Jetcapriest Website - Vercel

Static website for musician Jetcapriest - ready for Vercel deployment.

## Deploy to Vercel

### Option 1: GitHub Import (Recommended)
1. Push this folder to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "Add New" â†’ "Project"
4. Import your GitHub repo
5. Click "Deploy"

### Option 2: Vercel CLI
```bash
npm i -g vercel
cd Jetcapriest-Vercel
vercel
```

### Option 3: Drag & Drop
1. Go to [vercel.com/new](https://vercel.com/new)
2. Drag this folder onto the page

## Update Show Dates

Edit the events in `api/events.js` or `assets/js/main.js`
